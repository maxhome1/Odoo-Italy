Odoo-4-Italy
============

Odoo (formerly OpenERP) alternative Italian localization
--------------------------------------------------------

We are pursuing an alternative branch aiming the integration between EU rules and Italian fiscal law.

Versions
--------

**8.0** Odoo

**7.0** OpenERP
